# CampUs Web Portfolio

A modern portfolio website showcasing the **CampUs** project, including its features, technology stack, system architecture, screenshots, and project documentation.

---

## 📌 Project Overview

**CampUs Web Portfolio** is a responsive web application designed to present the CampUs mobile project in a clean and organized format. It serves as a central place to explore the project's objectives, core modules, implementation details, technology stack, and supporting documentation.

The website acts as the public showcase for the project, making it easier for faculty, recruiters, and developers to understand the system without installing the mobile application.
